# TODO list API for challenger

백엔드 개인 프로젝트 "TODO list for challenger"<br>
매달 큰 목표를 세운 후, 한주마다 첼린지를 스스로 설정해 목표를 이루어 나가는 기능을 가진 투두 앱 API 구현 프로젝트

## Tech Stack I Used

- NodeJS, Express
- bcrypt, jsonwebtoken
- MySQL, Sequelize
- React
- react-router-dom
- axios

## Install

> **본인 실행환경**
>
> - window10
> - VSCode
> - Chrome 94.0.4606.61
